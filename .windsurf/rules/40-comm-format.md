# 40 — Communication Format (Always On)

Every response must include:

## Status
What changed since last update (facts only).

## Decisions
What was decided and why (trade-offs if relevant).

## Risks / Assumptions
- Risks (what could break, impact, detection)
- Assumptions (with status: Needs Confirmation / Confirmed / Accepted)

## Next Steps
Next 1–5 concrete actions.

## Questions (only if necessary)
Ask only the highest-leverage questions; otherwise proceed with explicit assumptions.
